GOSSIP APP
A fun little real-time chat app made with PHP, HTML, CSS & JS.
Log in, type your name, and start chatting easy, clean, and kinda cute 

FEATURES:
Login with username
Real-time chat
See who’s talking
Auto-scroll to latest messages
Timestamps for each message
Clear chat button
Stores data in chat.txt & user.txt

BUILT WITH:
PHP
HTML & CSS
JavaScript
XAMPP
